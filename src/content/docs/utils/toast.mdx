## What are Toasts?
Toasts are used widely across Discord and other Websites/Services on certain Events like "Success" or "Failure" 
There are two options of toasts to use in Vencord. `copyWithToast` and `showToast`. 
The main difference are the usage between those two.

:::tip[What does that mean?]
Both Toasts still use `Toast` under the hood, but have different tasks from eachother.

`showToast` can be imported from `@webpack/common` (alias `src/webpack/common/index`)
`copyWithToast` can be imported from `@utils/misc` (alias `src/utils/misc`)
:::

### showToast()
The function expects following parameters,
```ts
message: string, 
type = ToastType.MESSAGE //ToastType.MESSAGE: 0,
options?: ToastOptions 
/* ToastOptions: position?: number; component?: React.ReactNode, duration?: number; */
```
Example:
```ts
showToast("Hello World")
showToast("Task xyz has failed", 2, { duration: 500 })
```

### copyWithToast()
Using this toast is especially recommended when interacing with a Users Clipboard.

This function expects following paramaters,
```ts
text: string, // The text that should be copied
toastMessage?: = "Copied to clipboard!" // What the toast displays
```
Example: 
```ts 
copyWithToast("Hello, World")
const shiggy = "shiggy"
copyWithToast(shiggy, `${shiggy} copied to clipboard`)
```
